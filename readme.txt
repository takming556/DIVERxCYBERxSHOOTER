■１．挨拶
　　この度は、「DIVERxCYBERxSHOOTER」を手にとって頂き、まことに有難
　う御座います。

　　本作品は、電子世界に於いて可愛らしいキャラクター達が、多種多様な
　弾幕を繰り広げるシューティングゲームになっています。

　　このような弾幕ＳＴＧは世間にありふれているわけではありません。し
　かし、このゲームジャンルを愛して止まない人が多くいることを私は知っ
　ています。そのようなコアなファンの方に満足いただけるかはわかりませ
　んが、繰り返しプレイをして、成長や達成感を味わっていただけますと幸
　いです。




■２．起動方法

　zipファイルを任意のディレクトリに展開します。
　DIVERxCYBERxSHOOTER.exeを開くとゲームが始まります。


■３．アンイストール方法

　フォルダごと削除してください。
